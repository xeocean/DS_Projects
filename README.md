# ML Projects

### Описание проектов

Проекты, выполненные в рамках обучения в Школе 21

|Project|Description|Nbviewer|
|-|-|-|
|DS1\_Tweets|Введение в обработку естественного языка: мешок слов, TF-IDF, стемминг, лемматизация, стоп-слова, косинусная близость, n-граммы, word2vec.||
|DS2\_Churn prediction|Введение в искусственные нейронные сети: полносвязные сети, скрытые слои, функции активации, обратное распространение ошибки, dropout.||
|DS3\_My Spotify|Введение в алгоритмы рекомендаций: неперсонализированные, контент-ориентированные, коллаборативная фильтрация.||
|DS4\_Understanding customer|Введение в глубокое обучение и NLP: рекуррентные нейронные сети (RNN), LSTM, Transformer, BERT.||
|DS5\_Uber|Введение в анализ временных рядов: стационарность, экспоненциальное сглаживание, SARIMA.||
|<br />DS6\_City life<br />|Введение в геопространственный анализ: GeoPandas, кластеризация, построение карт.||
|DS7\_Amazon|Введение в Apache Spark (Python API) и анализ сетей и графов.||
|DS8\_Fried Eggs|Введение в глубокое обучение и компьютерное зрение: CNN, сегментация объектов, детекция объектов, разметка данных.||
